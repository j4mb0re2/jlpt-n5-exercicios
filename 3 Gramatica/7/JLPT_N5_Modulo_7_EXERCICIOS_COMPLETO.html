❓ Complete com だれ ou だれの

Escolha entre だれ (quem) ou だれの (de quem) para completar a frase.

1. ______かばんですか。→ De quem é a mochila?
2. ______がきましたか。→ Quem veio?
3. これは______ほんですか。→ Este livro é de quem?
4. ______がいますか。→ Quem está aí?
5. あれは______ですか。→ Quem é aquilo?
6. これは______いぬですか。→ Este cachorro é de quem?
7. ______ねこですか。→ De quem é o gato?
8. ______がせんせいですか。→ Quem é o professor?
9. これは______おばさんですか。→ Esta é a tia de quem?
10. ______がたなかさんですか。→ Quem é Tanaka-san?
