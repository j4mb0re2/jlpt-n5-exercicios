📝 Traduza para o japonês

Escreva a frase em japonês usando o vocabulário aprendido. Clique no botão para ver se está certo.

1. Este livro é interessante.
2. Aquele carro é novo.
3. Esta cidade é silenciosa.
4. Tanaka-san é gentil.
5. Esta maçã não é gostosa.
6. Esse quarto não é limpo.
7. Café é mais amargo que chá.
8. O Japão é mais frio que o Brasil.
9. O celular é mais prático que o computador.
10. A televisão é mais entediante que os livros.

✅ Marque a alternativa correta

Escolha o adjetivo correto que completa a frase.

このくるまは______です。
2. Essa cidade não é silenciosa.
このまちは______です。
3. Este quarto é bonito.

4. Tanaka-san não é famoso.
たなかさんは______です。
5. Esse livro é chato.
そのほんは______です。
6. Essa comida não é gostosa.
このたべものは______です。
7. A biblioteca é calma.
としょかんは______です。
8. Este computador é prático.
このパソコンは______です。
9. Este restaurante não é bom.

10. Essa pessoa é simpática.

🔗 Complete com a partícula correta

Escolha entre は ou より conforme a função da frase.

1. Este livro é interessante.

2. Essa cidade é calma.
このまち______しずかです。
3. Tanaka-san é famoso.

4. Esse quarto não é bonito.

ブラジルのほうが にほん______あついです。
6. Celular é mais prático que computador.
スマホのほうが パソコン______べんりです。
7. TV é mais chata que livros.
テレビのほうが ほん______つまらないです。
