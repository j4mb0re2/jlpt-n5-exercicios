🧩 Complete a frase com a palavra correta

Escolha entre これ・それ・あれ・こちら ou as partículas は・の・か.

1. (______)はせんせいです。→ Este é o professor.
2. たなかさん(______)にほんじんです。→ Tanaka é japonês.
3. これはわたし(______)ほんです。→ Este é meu livro.
4. あなた(______)がくせいですか。→ Você é estudante?
5. (______)はともだちのかさです。→ Isso é o guarda-chuva do amigo.
6. (______)はがくせいのかばんです。→ Aquilo é a bolsa do estudante.
7. これはせんせい(______)じてんしゃです。→ Esta é a bicicleta do professor.
8. (______)はブラジルじんですか。→ Você é brasileiro?
9. かのじょ(______)しゅふです。→ Ela é dona de casa.
10. (______)はちゅうごくじんです。→ Isso é chinês.
